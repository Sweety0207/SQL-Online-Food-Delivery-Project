# SQL-Online-Food-Delivery-Project
This project focuses on analyzing a real world challenges in an online food delivery system using powerful querying capabilities of SQL
Using SQL query, we are able to conclude several meaningful insights such as customer behaviour, city wise trend, restaurant performance, product Preference, seasonal trends
Data driven insights can guide businesses in optimizing pricing, customer engagement, promotions and restaurant performance
Future scope include menu optimization, quality checks, more promotional discount and offers , delivery time etc
